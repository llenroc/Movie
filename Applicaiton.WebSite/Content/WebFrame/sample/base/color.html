Color

Metro UI CSS provides access to the main colors used in Windows 8. You can sets: fg-color-,bg-color-,border-color-,outline-color- for any elements.

  • *-color-green
  • *-color-greenDark
  • *-color-greenLight
  • *-color-pink
  • *-color-pinkDark
  • *-color-yellow
  • *-color-darken
  • *-color-purple
  • *-color-blue
  • *-color-blueDark
  • *-color-blueLight
  • *-color-orange
  • *-color-orangeDark
  • *-color-red
  • *-color-alizarin
  • *-color-white
